# EVALUATION STRATEGY

## Automatic Metrics

* ROUGE-1
* ROUGE-2
* ROUGE-L

## Ground Truth Strategy

* Create human-written summaries for 50–100 clusters

## Human Evaluation Criteria

1. Coherence
2. Coverage
3. Redundancy
4. Hallucination

## Known Limitation

ROUGE is not reliable when using pseudo-labels.
